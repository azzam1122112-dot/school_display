ضع لقطات الشاشة الحقيقية هنا ليستخدمها دليل البدء تلقائيًا.

الأسماء المعتمدة لكل صورة:

1. screens.png
- الصفحة: الشاشات
- المسار داخل اللوحة: dashboard/screen_list
- الأفضل أن تظهر: قائمة الشاشات أو زر إنشاء شاشة مع رابط الشاشة

2. school-data.png
- الصفحة: بيانات المدرسة
- المسار داخل اللوحة: dashboard/school_data
- الأفضل أن تظهر: الفصول والمواد والمعلمين

3. schedule.png
- الصفحة: الأيام وتوقيت الحصص
- المسار داخل اللوحة: dashboard/days_list
- الأفضل أن تظهر: الأيام الدراسية أو إعداد الحصص والتوقيت

4. settings.png
- الصفحة: الإعدادات
- المسار داخل اللوحة: dashboard/settings
- الأفضل أن تظهر: لون الثيم وخيار الكرت المميز

الامتدادات المدعومة لكل اسم:
- png
- webp
- jpg
- jpeg

أمثلة صحيحة:

- school-data.webp
